# Functional Requirements: Shopping List App

## Core Requirements

1. The application must allow users to add a new shopping list item by entering text and submitting it.
2. Each added item must be displayed in the shopping list.
3. Each displayed item must include:
   - A checkbox to mark the item as completed.
   - A visible pencil icon to indicate the item can be edited.
4. When a user checks an item's checkbox, the item name must display with a strikethrough style.
5. When a user unchecks an item's checkbox, the item name must return to normal (no strikethrough).
6. The application must allow users to edit an existing item's name (as indicated by the pencil icon).
7. Edited item names must be saved and immediately reflected in the list.
8. Item completion state (checked or unchecked) must remain associated with the correct item after edits.
9. The application must allow users to drag and drop items to reorder the shopping list.
10. After a drag-and-drop action, the list must immediately display items in the new order.
11. Item completion state (checked or unchecked) and item text must remain associated with the correct item after reordering.
